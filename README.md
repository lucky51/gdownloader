# gdownloader
a simple downloader

## Usage 

```shell
go install github.com/lucky51/gdownloader/cli/gd@latest
gd --url http://mirrors.aliyun.com/centos/7/isos/x86_64/CentOS-7-x86_64-DVD-2009.torrent --proxy socks5://127.0.0.1:10808 

```
